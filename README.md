
lievaart/alpine
===============

This ia a base alpine image slightly extended with some common packages / configuration.

some of the added packages:
* bash
* curl
* tree
* vim
* wget

usage
-----

to run the image:

`docker run -it lievaart/alpine bash`

refer to the alpine documentation for more info.
